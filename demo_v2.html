💡 How it works (V2 - Real Flow)
1. Your secret is used locally to derive a user_id (via Web Crypto API).
2. The server only sees the user_id and an encrypted challenge.
3. The server decrypts the challenge using Flash512.
4. If the challenge matches, you are authenticated.
The master secret NEVER leaves your browser.

🛡️ Powered by Flash512-Vanguard (server-side) · AES-256-GCM + Argon2id
